Identity/Coverage plots


In this dot plot the identity (y-axis) is compared with the coverage (x-axis)

Each red dot displays one blast-hit. The closer a point is to the right side, the more of the reference antibiotic resistance gene is covered by the input sequence represented by this dot. The closer a dot is to the top, the less differences there are in the alignment between the reference antibiotic resistance gene and the input sequence. 100% meaning the compared parts are identical.
